# Young Hearts Agent Platform
心青年智能体平台 - 单体仓库版本

## 快速启动
1. 安装所有依赖：`npm run install:all`
2. 同时启动前后端开发服务：`npm run dev`
3. 单独启动前端：`npm run dev:frontend`
4. 单独启动后端：`npm run dev:backend`
5. 构建生产版本：`npm run build`

## 目录结构
- `/frontend`：React前端代码
- `/backend`：后端服务代码
- `/config`：各环境统一配置
- `/docs`：项目文档
- `/shared`：前后端公共模块
